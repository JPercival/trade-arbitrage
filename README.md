# trade-arbitrage

Cross-chain arbitrage paper trader. Monitors real crypto prices across blockchains, detects pricing inefficiencies, and simulates trades — without risking real money.

See [VISION.md](./VISION.md) for full project specification.

## Quick Start

```bash
cp .env.example .env
# Add your Alchemy API keys and Telegram bot token
npm install
npm start
```

## Stack

Node.js · ethers.js · 1inch API · SQLite · Express · Railway
